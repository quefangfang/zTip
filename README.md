# zTip
提示消息插件 v1.0.0
提供 成功和失败两种颜色，暂不支持自定义颜色
### 参数列表 (options)

| 参数        | 说明       |默认值    | 可填值  |
| ------------|:-------------|:-------------:| -----:|
|msg           | 提示文字   |  提示消息  | string类型文字 |
|type          | 提示类型   |  success  | success、fail |
|time           | 文字消失间隔|   1000   |  毫秒为单位|
